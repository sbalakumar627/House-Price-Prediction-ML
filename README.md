# House Price Prediction using Machine Learning

This project predicts house prices using the California Housing Dataset.

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## Techniques Applied
- Data Preprocessing
- Feature Scaling
- Model Training
- Model Evaluation

## Evaluation Metrics
- RMSE
- R² Score

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
